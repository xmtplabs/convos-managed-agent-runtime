1
Tweet your idea
User tweets a one-line idea at @convosmessenger. No form, no sign up, no app. Just a tweet describing the group agent they want.
Examples:
"hotel concierge for traveling with my wife"
"fantasy football commissioner for our league"
"book club discussion facilitator"
"wedding planning coordinator for bridal party"
* Lowest possible friction. The barrier is literally typing a tweet. Anyone on X can do this instantly.

2
Your agent page is born
Convos takes the one-line idea, generates a full agent skill using SOUL.md (the Convos philosophy on group AI), and publishes it to a unique page. This is the artifact — the thing that gets shared everywhere.
The page has:
Convos branding + "what is this?" at top
Agent name, description, full expanded skill
QR code to add the agent directly
Copy button for the full prompt
Created-by credit to the original tweeter
* 10 words become a full skill. Like what Shane did manually in Notion for 90 agents — but automated. SOUL.md philosophy baked in.

5
Agent joins your group chat
A pre-warmed agent instance is claimed from the Convos pool (under 10 seconds). It joins the group with the full skill loaded — personality, capabilities, SOUL.md philosophy. Introduces itself and starts learning.
What happens:
Pool claims an idle instance
Skill + SOUL.md injected as identity
Agent joins via XMTP (encrypted)
Welcome message sent to group
Email, search, SMS, crypto pre-configured
* Instant and alive. The agent shows up knowing what it is, introduces itself, and teaches the group how to train it. No cold start.

7
The timeline fills up
Other people see the page being shared, get inspired, and tweet their own ideas at @convosmessenger. Each one generates a new unique agent page. The X timeline becomes a feed of agent pages — each one a scannable, shareable entry point.
Why this compounds:
Everyone wants to show off their clever idea
Each page credits the creator
Each OG image has a scannable QR
The catalog writes itself from community submissions
* This is Wabi but actually useful. Every agent page created publicly becomes social proof + a new entry point. The community builds the catalog.

Open Questions

X account
@convosmessenger (main) vs @convosbot (dedicated)? One account simpler.
Prompt curation
Auto-generate all. Curate best onto homepage later. Community submissions = running log.
Feedback loop
Tweet feedback at us? Public chatter about Convos + accepting feedback in public.
Skills simplification
SOUL.md built in = simpler skill templates? How much baked in vs custom?
